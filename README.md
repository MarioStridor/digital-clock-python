# Digital Clock

A real-time digital clock built with Python and Tkinter.
Displays current time and date, refreshing every second.

## Features
- Live clock updating every second
- Displays time in 12-hour format with AM/PM
- Shows current date alongside the time
- Clean GUI with bold, large font display

## Tech Stack
- Python
- Tkinter
- time module (strftime)

## How to Run
1. Make sure Python is installed
2. Run: python digital_clock.py
